# Парсер рейтингов фильмов с KinoAfisha.info 

## Постановка задачи
Цель проекта: Разработка автоматизированного инструмента для сбора и анализа данных о фильмах с сайта KinoAfisha.info.

Проблема, которую решает проект:

- Ручной сбор информации о фильмах (названия, жанры, рейтинги, кассовые сборы) — трудоемкий и времязатратный процесс
- Отсутствие удобного способа получения структурированных данных для анализа кинематографических тенденций
- Необходимость в автоматизированном мониторинге изменений в рейтингах фильмов 

## Установка и запуск 

1. Клонирование репозитория
   git clone https://github.com/pacelzak/parser.git 
2. Установка необходимых библиотек:
   pip install requests beautifulsoup4 lxml pandas openpyxl
## Базовый запуск 
1. Запустите скрипт из командной строки
   python parser.py
2. Введите тип рейтинга, когда появится запрос
3. Скрипт начнет сбор данных. Прогресс будет отображаться в консоли
4. Результаты будут сохранены в файл
